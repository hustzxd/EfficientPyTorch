# EfficientPyTorch

## [Install](docs/install.md)
```bash
git clone https://github.com/hustzxd/EfficientPyTorch.git

cd EfficientPyTorch
export PYTHONPATH=$PYTHONPATH:`pwd`
pip install -r requirements.txt
```

- [x] [Learned Linear Symmetric Quantization (LLSQ)](docs/llsq.md)
